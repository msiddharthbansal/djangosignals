By default, Django signals are executed synchronously. 

1) Create a Django project and app.
2) Register the app to INSTALLED_APPS in config/settings.py .
3) Define a Model and a Signal in myapp/models.py .
(File shared as Q1/models.py)
4) Apply migrations.
5) Make a file to test signals in root directory and execute it.
(File shared as Q1/test_signals.py)
